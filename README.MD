## Sobre o Projeto

Este projeto acadêmico apresenta uma implementação em Java de um **Sistema Acadêmico**, desenvolvido como parte da disciplina de Programação Orientada a Objetos. O sistema modela os principais componentes: cadastro de aluno, professor, cadastro de disciplina, cadastro de estágio, cadastro de aluno em disciplina, cadastro de aluno em estágio, registro de notas, registro de avaliação em estágio, cálculo de resultado do aluno em componentes acadêmicos, situação do aluno, listagem de disciplinas e estágios e consulta de todos os alunos matriculados.  

## Contexto Acadêmico

Este trabalho foi desenvolvido como atividade avaliativa para a disciplina de Programação Orientada a Objetos do curso de Tecnologia em Sistemas para Internet do IFPB. O objetivo é consolidar o aprendizado dos conceitos de POO através de um projeto prático que simula cenários reais da área de redes de computadores.
